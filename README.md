# Simple Interest Calculator

## Description
This project is a simple Bash script that calculates the **Simple Interest** based on user input.

The user provides:
- Principal amount
- Rate of interest
- Time period

The script then calculates the simple interest using the formula:

SI = (P × R × T) / 100

Where:
P = Principal amount  
R = Rate of interest  
T = Time period  

## Features
- Accepts user input from the terminal
- Calculates simple interest instantly
- Simple and easy Bash implementation

## Example

Principal = 1000  
Rate = 5  
Time = 2  

Simple Interest = 100
